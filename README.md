# Cat Shelter 2026

A small Node.js project developed as part of the SoftUni JS Back-End course.

The purpose of this project is to build a web server using only Node.js core modules in order to better understand the client-server relationship and the fundamentals of back-end development.

## Overview

**Cat Shelter** is a simple web application that allows users to browse cats available for adoption.

The application supports full CRUD operations:

- Create new cats
- Read and view all available cats
- Update existing cat information
- Delete cats when they find a new home

Additionally, the application includes a search functionality for finding cats more easily.

## Data Storage

This project was created for educational purposes only, therefore **no database is used**.

All application data is stored in memory and is reset every time the server restarts.

## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd cat-shelter-2026
```

3. Install dependencies (if any):

```bash
npm install
```

4. Start the server:

```bash
npm start
```

5. Open your browser and navigate to:

```text
http://localhost:3000
```
